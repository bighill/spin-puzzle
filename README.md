# Spin Puzzle

Simple puzzle game.

[Play the Demo](https://bighill.github.io/spin-puzzle/)

## Stack

- Vanilla HTML/JS/CSS
- Typescript
- Vite

## Usage

```bash
git clone https://github.com/bighill/spin-puzzle
cd spin-puzzle
npm install
```

```bash
npm run dev
npm run build
```

> The build command will build to `/docs/` so that the demo is available in Github Pages.
